Double Tap Gesture Delegate

  • mapController(_:recognizer:shouldRecognizeDoubleTapGesture:)

    Asks the delegate if the map should recognize this double tap and perform default functionality (which is nothing, currently).

    Declaration

    Swift

    func mapController(_ controller: MapViewController, recognizer: UIGestureRecognizer, shouldRecognizeDoubleTapGesture location: CGPoint) -> Bool

    Parameters

    controller

    The MapViewController that wants to recognize the tap.
    recognizer

    The recognizer that initially recognized the tap.
    location

    The screen coordinates that the tap occured in relative to the bounds of the map.

    Return Value

    Return true for default functionality, or false if don’t want it recognized (or plan on handling it yourself).
